# 拾光书屋系统测试文档

## 测试环境

| 项目 | 说明 |
|------|------|
| 前端 | Vue 3 + Vite + Element Plus，端口 5173 |
| 后端 | Spring Boot 3.2.5 + MyBatis-Plus 3.5.6，端口 8081 |
| 数据库 | MySQL 8.0 |
| AI 服务 | MiniMax-M2.7（api.minimaxi.com） |
