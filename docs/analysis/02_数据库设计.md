# 网上书店系统数据库设计

---

## 1. 核心数据表概述

| 表名 | 说明 |
|------|------|
| `user` | 用户表 |
| `book` | 书籍表（含 orig_price 划线原价、quote 名言、favorited_count 收藏数） |
| `category` | 分类表 |
| `author` | 作者表（bio/国籍/生卒年/奖项） |
| `book_chapter` | 目录表（章节 + 页码） |
| `cart` | 购物车表 |
| `order` | 订单表 |
| `order_item` | 订单明细表 |
| `address` | 收货地址表 |
| `review` | 评论表 |
| `payment` | 支付记录表 |
| `favorite` | 收藏表（user_id + book_id 唯一约束，含 favorited_price 收藏时价格） |

---

## 2. 数据表结构

### 2.1 用户表 (user)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 用户ID |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 用户名 |
| password | VARCHAR(64) | NOT NULL | 密码（加密） |
| email | VARCHAR(100) | UNIQUE | 邮箱 |
| phone | VARCHAR(20) | UNIQUE | 手机号 |
| role | VARCHAR(20) | DEFAULT 'user' | 角色（user/admin） |
| status | TINYINT | DEFAULT 1 | 状态（1正常/0禁用） |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 2.2 书籍表 (book)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 书籍ID |
| isbn | VARCHAR(20) | UNIQUE | ISBN号 |
| title | VARCHAR(200) | NOT NULL | 书名 |
| author | VARCHAR(100) | NOT NULL | 作者 |
| author_id | BIGINT | FK | 作者ID（关联author表） |
| publisher | VARCHAR(100) | | 出版社 |
| publish_date | DATE | | 出版日期 |
| price | DECIMAL(10,2) | NOT NULL | 价格（现价） |
| orig_price | DECIMAL(10,2) | NULL | 划线原价 |
| stock | INT | DEFAULT 0 | 库存 |
| sales | INT | DEFAULT 0 | 销量 |
| favorited_count | INT | DEFAULT 0 | 收藏数（原子SQL更新） |
| category_id | BIGINT | FK | 分类ID |
| description | TEXT | | 简介 |
| cover_url | VARCHAR(500) | | 封面图 |
| status | TINYINT | DEFAULT 1 | 状态（1上架/0下架/2预售/3即将上架） |
| quote | VARCHAR(500) | | 书中摘抄名言 |

### 2.8 订单表 (`order`)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 订单ID |
| order_no | VARCHAR(32) | UNIQUE, NOT NULL | 订单号 |
| user_id | BIGINT | FK, NOT NULL | 用户ID |
| total_amount | DECIMAL(10,2) | NOT NULL | 总金额 |
| status | VARCHAR(20) | NOT NULL | 订单状态 |
| pay_status | VARCHAR(20) | DEFAULT 'unpaid' | 支付状态 |
| address_id | BIGINT | FK | 收货地址ID |
| receiver_name | VARCHAR(50) | | 收货人 |
| receiver_phone | VARCHAR(20) | | 联系电话 |
| receiver_address | VARCHAR(300) | | 收货地址 |
| remark | VARCHAR(500) | | 备注 |
| payment_id | BIGINT | | 支付记录ID |
| pay_time | DATETIME | | 支付时间 |
| express_no | VARCHAR(50) | | 物流单号 |
| expire_time | DATETIME | | 订单过期时间 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

**订单状态流转：**
```
created → paying → paid → shipped → delivered → completed
   ↓ (过期)    ↓ (取消)
expired      cancelled
                    paid/shipped → refunded
                    delivered/completed → after_sale
```
- created/paying 可取消，created 可过期，paid/shipped 可退款，delivered/completed 可售后

## 3. 表关系（ER图描述）

```
user (1) ────── (n) address
   │
   ├── (1) ────── (n) cart ────── (1) book
   │
   ├── (1) ────── (n) favorite ────── (1) book
   │
   ├── (1) ────── (n) order ────── (n) order_item ────── (1) book
   │              │
   │              └── (1) ────── (1) payment
   │
   ├── (1) ────── (n) review ────── (1) book
   │
   └── ──────────────────────────── category (1) ────── (n) book

author (1) ────── (n) book
book (1) ────── (n) book_chapter
```

## 5. 索引设计

| 表名 | 索引字段 | 类型 | 说明 |
|------|---------|------|------|
| user | username | UNIQUE | 登录查询 |
| user | email | UNIQUE | 邮箱查询 |
| book | isbn | UNIQUE | ISBN查询 |
| book | category_id | INDEX | 分类筛选 |
| book | title | INDEX | 标题搜索 |
| cart | user_id + book_id | UNIQUE | 防止重复加入 |
| favorite | user_id + book_id | UNIQUE | 防止重复收藏 |
| order | order_no | UNIQUE | 订单号查询 |
| order | user_id + create_time | INDEX | 用户订单列表 |
| order | expire_time + status | INDEX | 订单过期查询 |
| payment | payment_no | UNIQUE | 支付流水号查询 |
| payment | order_id | INDEX | 订单支付查询 |
| review | book_id | INDEX | 书籍评论列表 |
