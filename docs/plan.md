## 1. 设计总纲

### 1.1 设计定位

**「走进一家深夜还亮着暖光的老派独立书店」**

拾光书屋的视觉语言不是现代极简科技风，也不是可爱卡通风，而是**经典木质书店美学**——深色胡桃木书架、泛黄纸张的温度、黄铜台灯的光晕。每一个界面都应让用户产生「在安静的书店里翻书」的沉浸感。

### 1.2 设计原则

| 原则 | 说明 | 反面案例 |
|------|------|----------|
| **温暖而非刺眼** | 所有强调色使用古铜金 `#C09A4B`，而非亮黄 `#FFB300`。金色像灯光晕染，不做成霓虹灯 | 电商大促风格的荧光黄按钮 |
| **厚重而非沉重** | 大面积使用羊皮纸底色 `#EDE6D6`，深色区域用 `#1C120C`～`#2E1F15` 多级递进 | 纯黑背景 + 白字 |
| **书卷气而非老气** | 衬线字体（思源宋体）用于标题，无衬线（思源黑体）用于正文。排版有呼吸感 | 全局使用系统默认字体 |
| **克制而非匮乏** | 金色只用在最精准的 5% 位置：价格、角标、选中态、下划线 | 金色大面积铺满 |
| **留白而非空洞** | 书卡之间保持 24px 间距，区域之间保持 80px 呼吸空间 | 所有内容挤在一起 |

### 1.3 页面间导航关系

```
首页
 ├── 全部书籍（主商城页，所有书籍的完整列表）
 ├── 分类浏览（按类别组织的树状导航 + 筛选）
 ├── 排行榜（多维度榜单：畅销 / 评分 / 新锐 / 借阅）
 ├── 新书上架（时间线式的新书展示 + 预售）
 └── 关于我们（品牌故事 / 团队 / 理念 / 联系方式）
```

## 2. 色彩系统

### 2.1 主色板

```css
:root {
  /* ── 主色：胡桃木色系 ── */
  --color-primary:       #4A3526;   /* 主色，按钮、标签 */
  --color-primary-mid:   #5C4434;   /* 中间态，hover 过渡 */
  --color-primary-light: #7A6252;   /* 浅色文字、图标 */
  --color-primary-dark:  #2E1F15;   /* 深色区域背景 */
  --color-primary-abyss: #1C120C;   /* 最深，导航栏、页脚 */

  /* ── 背景：羊皮纸色系 ── */
  --color-bg:        #EDE6D6;  /* 页面主底色 */
  --color-bg-warm:   #E5DDD0;  /* 次级背景、区块底 */
  --color-bg-card:   #F5F0E8;  /* 卡片底色 */
  --color-bg-cream:  #F8F4ED;  /* 最浅，hover 态底色 */

  /* ── 强调色：古铜金色系 ── */
  --color-accent:        #C09A4B;  /* 主强调色 */
  --color-accent-light:  #D4B06A;  /* 浅金色，大数字 */
  --color-accent-muted:  #A68638;  /* 暗金色，价格文字 */
  --color-accent-glow:   rgba(192,154,75,0.12);  /* 光晕 */

  /* ── 文字 ── */
  --color-text:            #2B1D14;  /* 主文字，墨棕 */
  --color-text-secondary:  #6B5B4E;  /* 次要文字 */
  --color-text-light:      #9B8B7E;  /* 辅助文字、占位符 */
}
```

### 2.2 色彩使用规则

| 场景 | 颜色 | 使用位置 |
|------|------|----------|
| 页面底色 | `--color-bg` `#EDE6D6` | `body` 背景 |
| 卡片/面板 | `--color-bg-card` `#F5F0E8` | 书卡、侧栏、表单 |
| 深色区域 | `--color-primary-abyss` `#1C120C` | 导航栏、页脚、Hero |
| 主按钮 | 渐变 `accent → accent-muted` | CTA 按钮 |
| 价格文字 | `--color-accent-muted` `#A68638` | 书籍定价 |
| 选中/激活态 | `--color-primary` `#4A3526` | 分类标签选中、Tab 选中 |
| 分割线 | `rgba(74,53,38,0.1)` | 卡片边框、区域分隔 |
| 阴影 | `rgba(44,30,20,0.08~0.25)` | 卡片投影 |

### 2.3 金色使用禁区

> **金色（`--color-accent`）全站占比不超过 5%。**

- ✅ 允许：价格、徽章角标、评分星星、选中下划线、Hero 装饰
- ❌ 禁止：大面积背景、正文文字、导航文字、卡片边框、按钮边框

## 4. 组件规范

### 4.1 书籍卡片（Book Card）

书籍卡片是全站使用频率最高的组件，共两种形态：

#### 网格形态（默认）

```
┌─────────────────┐
│                  │  ← 3:4 封面图/色块
│    [角标]    [♡] │
│                  │
│     书名标题      │
│                  │
├─────────────────┤
│ 书名（最多2行）   │  ← Serif 字体
│ 作者             │  ← 浅色辅助文字
│ ¥45.60  ★9.4    │  ← 价格金色 + 评分
└─────────────────┘
```

- 封面宽高比 `3:4`
- 悬停上浮 5px + 阴影加深 + 封面微缩放 1.04x
- 收藏按钮（♡）悬停时才显示，点击切换 ♥

#### 列表形态

```
┌──────────┬──────────────────────────────┐
│          │  书名                         │
│  封面    │  作者                         │
│  3:4     │  简介文字（最多2行）           │
│          │  ¥45.60          ★9.4        │
└──────────┴──────────────────────────────┘
```

- 左侧封面 120px 宽，右侧展示更多信息
- 通过工具栏的视图切换按钮控制

### 4.2 分类标签（Category Pill）

```
┌──────────┐  ┌──────────┐  ┌──────────┐
│  全部     │  │ 文学小说  │  │ 科技新知  │
└──────────┘  └──────────┘  └──────────┘
  (选中态)       (默认态)       (默认态)
```

- 默认态：白底 + 细边框，文字 `--color-text-secondary`
- 悬停态：边框变深，底色微变
- 选中态：`--color-primary` 底色 + 浅色文字

### 4.3 按钮系统

| 按钮类型 | 样式 | 用途 |
|----------|------|------|
| `.btn-primary` | 古铜金渐变底 + 深色文字 | 主要操作（立即购买、开始探索） |
| `.btn-secondary` | 透明底 + 浅色文字 + 细边框 | 次要操作（加入书架、本周书单） |
| `.btn-ghost` | 无边框无底色 + 文字色 | 最轻量操作（查看全部 →） |
| `.btn-dark` | 深色底 + 浅色文字 | 深色区域内的操作按钮 |

### 4.4 分页器（Pagination）

```
‹ 上一页  [1]  2   3   4   5  ...  186  下一页 ›
```

- 选中页：`--color-primary` 底色 + 浅色文字
- 普通页：白底 + 细边框
- 不可用：降低透明度 0.35 + 禁止指针

### 4.5 评分组件

- 五角星使用 `--color-accent` `#C09A4B`
- 数值使用 `--color-text-light`
- 格式：`★★★★☆  8.9`

### 5.3 功能清单

#### 5.3.1 快速筛选标签栏

位于页面标题下方，工具栏上方。提供一键式快捷筛选。

| 标签 | 功能 |
|------|------|
| 全部 | 清除所有快捷筛选 |
| 今日特价 | 筛选今日参与促销的书籍 |
| 限时折扣 | 筛选折扣率 > 0 的书籍 |
| 全场包邮 | 筛选包邮商品 |
| 套装优惠 | 筛选套装商品 |
| 预售新品 | 筛选预售状态的书籍 |
| 电子书 | 筛选电子书格式 |

**交互说明：**
- 标签可多选，选中后高亮（`--color-primary` 底色）
- 选中状态同时反映在工具栏的「已选筛选标签」中
- 标签数量可横向滚动（移动端），PC 端全部展示

#### 5.3.2 工具栏

```
┌────────────────────────────────────────────────────────────┐
│  共 128,000 本  │ [今日特价 ✕][包邮 ✕]  清除全部 │ 排序 ▾ │ ▦ ☰ │
└────────────────────────────────────────────────────────────┘
```

**左侧：**
- 结果计数：`共 <strong>128,000</strong> 本`
- 活跃筛选标签：胶囊形，点击 ✕ 移除
- 「清除全部」：文字链接，虚线下划线

**右侧：**
- 排序下拉：自定义样式 `<select>`，选项包括：
  - 综合推荐（默认）
  - 销量从高到低
  - 价格从低到高
  - 价格从高到低
  - 评分从高到低
  - 出版时间从新到旧
  - 评论数从多到少
- 视图切换：网格 ▦ / 列表 ☰

#### 5.3.3 书籍网格

- **PC 端**：5 列网格，间距 24px
- **平板端**：3 列
- **移动端**：2 列，间距 14px
- 卡片进入视口时触发 `fadeUp` 动画，每张卡片错开 30ms

**列表视图：**
- 1 列全宽，左侧封面 120px，右侧展示书名 + 作者 + 2 行简介 + 价格评分
- 封面高度 170px

#### 5.3.4 无限滚动 / 分页（二选一）

**方案 A — 传统分页：**
- 页码组件，每页展示 20 本
- 首页/末页快捷跳转

**方案 B — 无限滚动（推荐）：**
- 首次加载 20 本
- 滚动至底部 200px 时自动加载下一批
- 加载中显示木质书纹旋转动画
- 加载完所有数据后显示「已经到底了，去别处逛逛吧」
- URL 保持 `?page=N` 参数，支持分享和后退

### 5.4 空状态设计

- 滚动时日期标题 sticky 到顶部（短暂吸附后被下一个日期推出）

### 9.3 功能清单

#### 9.3.1 Hero 品牌宣言

**设计要点：**
- 全屏深色背景（`--color-primary-abyss`）
- 背景叠加纸张肌理暗纹（`opacity 0.03`）
- 中央大标题：Serif 字体，3rem，`--color-bg-warm`
- 副标题：Sans 字体，1.05rem，`--color-text-light`
- 无按钮，纯文案表达——这是一个「感受」区域，不是「行动」区域
- 视差滚动效果：背景滚动速度比前景慢 50%

**文案方向：**
- 标题用一句有态度的话定义品牌（如「在快时代，做一家慢书店」）
- 副标题用 1~2 句话展开

#### 9.3.2 品牌故事

**布局：左图右文**
- 左侧：模拟旧照片/书店场景图（可用色块替代，带 sepia 色调和圆角）
- 右侧：Serif 标题 + Sans 正文，段落之间留足呼吸空间
- 滚动进入视口时，图片从左侧 `slideIn`，文字从右侧 `slideIn`

**文案结构：**
1. **起源**：我们为什么做这件事（问题 → 行动）
2. **理念**：我们怎么做的（坚持人工选书、深度阅读）
3. **愿景**：我们要去哪里（让好书触手可及）

#### 9.3.3 核心理念

**三列等分卡片：**
- 每张卡片：图标 + 标题 + 说明文字
- 图标使用线描风格（不是 emoji），颜色 `--color-accent`
- 卡片底色 `--color-bg-card`，细边框
- 悬停时边框变金色 + 微上浮

| 理念 | 标题 | 说明 |
|------|------|------|
| 📖 | 人工选书 | 每一本上架的书都经过编辑团队的阅读和讨论 |
| 🔍 | 深度阅读 | 提供书评、导读和主题书单，不只卖书，更陪读书 |
| 🤝 | 读者社区 | 线下读书会、线上讨论社群，让阅读不再孤独 |
| 🚚 | 品质服务 | 精品包装、极速配送、无忧退换 |

## 10. 全局交互规范

### 10.1 页面过渡

| 交互 | 效果 | 时长 | 缓动 |
|------|------|------|------|
| 卡片进入视口 | `fadeUp` 20px → 0 | 0.5s | `ease` |
| 卡片悬停 | `translateY(-5px)` + 阴影加深 | 0.4s | `cubic-bezier(0.25,0.46,0.45,0.94)` |
| 按钮悬停 | `translateY(-2px)` + `letter-spacing` 展开 | 0.3s | `ease` |
| 收藏点击 | 颜色切换 + `scale(1.1) → scale(1)` | 0.2s | `ease` |
| 筛选标签移除 | `opacity(0) + scale(0.8)` → 移除 DOM | 0.2s | `ease` |
| Tab 切换 | 内容 `opacity(1) → 0` → 数据更新 → `0 → 1` | 0.3s | `ease` |

### 10.2 阴影层级

| 层级 | 值 | 使用场景 |
|------|-----|----------|
| 静止 | `0 2px 8px rgba(44,30,20,0.06)` | 默认卡片 |
| 悬停 | `0 16px 40px rgba(44,30,20,0.16)` | 卡片 hover |
| 浮层 | `0 20px 60px rgba(30,18,10,0.25)` | 弹窗、下拉菜单 |
| 导航 | `0 1px 20px rgba(30,18,10,0.25)` | 固定导航栏 |

### 10.3 加载状态

**骨架屏：**
- 卡片区域显示等比例骨架块
- 骨架块底色 `--color-divider`，高光动画从左到右扫过
- 每个骨架块保持 3:4 宽高比

**加载中：**
- 无限滚动加载时，底部显示一个旋转的书页图标
- 图标旋转速度 1.5s/圈

**加载失败：**
- 显示「加载出了点问题，点击重试」+ 重试按钮

### 10.4 Toast 提示

```
┌──────────────────────────────┐
│  ✓ 已加入购物车               │  ← 操作成功
└──────────────────────────────┘
```

- 出现在页面顶部中央
- 背景 `--color-primary-dark`，文字 `--color-bg-warm`
- 成功：左侧绿色圆点
- 错误：左侧红色圆点
- 自动消失：3 秒后 `fadeUp` 退出
