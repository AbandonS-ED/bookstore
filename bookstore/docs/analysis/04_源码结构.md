# 网上书店系统源码目录结构

> 本文档仅展示项目结构，不含具体代码

---

## 1. 后端项目结构（Spring Boot）

```
bookstore/
│
├── pom.xml                                    # Maven项目配置
├── .gitignore                                 # Git忽略配置
│
└── src/
    └── main/
        ├── java/
        │   └── com/example/bookstore/       # 主包
        │       │
        │       ├── BookstoreApplication.java    # 启动类（@MapperScan）
        │       │
        │       ├── config/                    # 配置包
        │       │   ├── CorsConfig.java       # 跨域配置（localhost:*）

        │       │       ├── OrderServiceImpl.java
        │       │       ├── PaymentServiceImpl.java
        │       │       ├── AddressServiceImpl.java
        │       │       ├── ReviewServiceImpl.java

        │       │
        │       ├── mapper/                    # 数据访问层（12个）

        │       │   ├── FavoriteVO.java
```

## 3. 数据库脚本位置

```
sql/
├── init.sql                        # 建表脚本（12张表 + 种子数据）
│   ├── 用户表 user
│   ├── 分类表 category
│   ├── 书籍表 book（含划线原价、名言、收藏数、预售日期）
│   ├── 作者表 author
│   ├── 目录表 book_chapter
│   ├── 收藏表 favorite
│   ├── 收货地址表 address
│   ├── 购物车表 cart
│   ├── 订单表 `order`
│   ├── 订单明细表 order_item
│   ├── 支付记录表 payment
│   └── 评论表 review
```

**注意**：`order` 是 MySQL 保留字，表名需用反引号 `` `order` `` 包裹。

## 4. 关键配置文件位置

| 文件 | 路径 | 说明 |
|------|------|------|
| Maven配置 | `pom.xml` | 项目根目录 |
| 后端主配置 | `src/main/resources/application.yml` | 数据库、端口、JWT等 |
| 后端本地配置 | `src/main/resources/application-local.yml` | 数据库密码、callback密钥（gitignore排除） |
| 前端配置 | `bookstore-frontend/vite.config.js` | Vite构建配置，代理/api、/admin-api、/pictures到8081 |
| 前端入口 | `bookstore-frontend/index.html` | Vue应用入口 |
| 封面图片 | `pictures/` | 书籍封面JPG图片，通过`/pictures/*`访问（后端静态映射+Vite代理） |

## 5. 包命名规范

| 包名 | 职责 | 示例 |
|------|------|------|
| `com.example.bookstore` | 根包 | 项目主包 |
| `com.example.bookstore.config` | 配置类 | 框架配置 |
| `com.example.bookstore.interceptor` | 拦截器 | 认证/授权 |
| `com.example.bookstore.controller` | 控制层 | 用户请求入口 |
| `com.example.bookstore.service` | 业务层 | 业务逻辑处理 |
| `com.example.bookstore.mapper` | 数据访问层 | 数据库操作 |
| `com.example.bookstore.entity` | 实体类 | 数据库表映射 |
| `com.example.bookstore.dto` | 数据传输对象 | 层间数据传递 |
| `com.example.bookstore.vo` | 视图对象 | 返回前端数据 |
| `com.example.bookstore.common` | 通用类 | 工具类、常量 |
| `com.example.bookstore.exception` | 异常处理 | 统一异常处理 |
| `com.example.bookstore.util` | 工具类 | 密码/JWT/上下文 |
